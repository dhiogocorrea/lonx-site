Combat math

Formulae from the original system

These preserve the draft's mathematical direction while making the presentation easier to use for balancing. Abbreviations: WPD = weapon physical damage, MWPD = magic weapon damage, AD = arrow damage, BDMG = bomb base damage, LVL = character level.

Offense

Attack & damage

SystemFormula / Rule
Base ATK(STR x 0.1) + WPD
Base MATK(WIS x 0.1) + MWPD
Weapon Mastery RuleNo mastery = 30% of weapon damage; Mastery Level I = 50%; each level adds +5% until Level Master = 100%.
Melee Damage(STR / 11) x ATK, then each hit varies by roughly 12%.
Ranged Damage[(DEX / 11) x WPD] + [AD x (DEX / 3)]
Bomb Damage[(BDMG + INT) x 30 / 11]
Magic Damage[(MATK x 60 / 11) + (INT x 20 / 11)] x WIS / 36; if WIS < 36, use 36.
Critical Hit RateStarts at 2% and increases by 0.083% for each point in LUCK.
Defense & survival

HP, resistance & capacity

SystemFormula / Rule
HP(6 x STR + 30 x CON + 4 x LVL) / 1.036
Carrying Capacity[100 + (CON / 10) + (STR / 2)] / 1.036
Natural Resistance(CON / 3) + (0.03 x LVL), applied to pure damage such as critical strikes.
Knockback ResistanceEvery 7 CON grants 1% knockback resistance.
Tempo & resources

Speed & mana

SystemFormula / Rule
Attack SpeedBase ASPD = [(AGI - 10) x 0.3125] + 24; weapon speed modifies final ASPD; max is 165.
Mana[(200 x INT / 11) + (0.9 x LVL)]
Notes

Balancing guidance

  • Race multipliers (see the Races page) scale how quickly each stat feeds these formulae.
  • Magic damage floors WIS at 36, so casters below that threshold gain nothing extra from the WIS scalar.
  • The / 1.036 divisor in HP and Carrying Capacity is a global tuning constant; adjust it to rescale survivability server-wide.
  • Critical rate scales purely from LUCK, which is why Humans (1.20x LUCK) crit most reliably and Phantoms (0.78x) least.
